Perguntas Frequentes (FAQ)

O que esta ferramenta faz?

Esta ferramenta é uma interface web que permite analisar arquivos e URLs em busca de ameaças de segurança. Ela utiliza APIs de serviços renomados como VirusTotal e Tria.ge, além de IAs Generativas (Google Gemini ou OpenAI) para fornecer um resumo claro dos riscos.

Preciso de chaves de API para usar a ferramenta?

Sim. Para que as análises externas funcionem, você precisa fornecer as chaves de API dos serviços. A análise local (identificação de tipo de arquivo e strings) funciona sem chaves, mas é muito limitada.

Onde consigo as chaves de API?

Você pode obtê-las gratuitamente nos sites oficiais de cada serviço:

  • VirusTotal: virustotal.com/gui/my-apikey
  • Tria.ge: tria.ge/account
  • Google Gemini: aistudio.google.com/app/apikey
  • OpenAI (ChatGPT): platform.openai.com/api-keys

Como eu configuro as minhas chaves na aplicação?

As chaves de API podem ser configuradas diretamente na interface da aplicação. Na página inicial, clique em "Configurações" para expandir a seção e insira suas chaves nos campos apropriados. As chaves são salvas automaticamente e a aplicação será recarregada para aplicar as alterações.

A aplicação não abre e o navegador mostra "localhost recusou a conexão". O que fazer?

Este erro geralmente significa que o servidor não conseguiu iniciar. Verifique a janela do terminal que o iniciar.bat abriu. Ela deve conter uma mensagem de erro. As causas mais comuns são:

  • Uma ou mais dependências não foram instaladas corretamente. Tente executar pip install -r requirements.txt na pasta app.
  • As chaves de API no arquivo .env podem estar em um formato inválido. Execute o configurar_chaves.py novamente para garantir que estão corretas.

É seguro enviar meus arquivos para esta ferramenta?

Sim. A análise de arquivos e URLs é executada em ambientes seguros e remotos (sandboxes) fornecidos pelas APIs (VirusTotal, Tria.ge), e não na sua máquina local. O arquivo é enviado diretamente para eles.
